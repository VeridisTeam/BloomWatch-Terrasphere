API para Desenvolvedores

Integre dados de florações de algas nocivas em suas aplicações com nossa API RESTful robusta e bem documentada.

Endpoints Principais

GET /v1/blooms/current

Florações Atuais

Retorna todas as florações de algas ativas no momento.

Exemplo de Resposta:
{
  "data": [
    {
      "id": "hab_2024_001",
      "location": {
        "lat": -23.5505,
        "lng": -46.6333,
        "name": "Costa de São Paulo"
      },
      "species": "Alexandrium minutum",
      "toxin_level": 2.4,
      "severity": "moderate",
      "detected_at": "2024-01-15T10:30:00Z"
    }
  ],
  "total": 127,
  "page": 1
}
GET /v1/blooms/historical

Dados Históricos

Acesse dados históricos de florações com filtros avançados.

Parâmetros:
  • start_date - Data inicial (ISO 8601)
  • end_date - Data final (ISO 8601)
  • region - Região geográfica
  • species - Espécie de alga
GET /v1/predictions

Previsões

Obtenha previsões de florações baseadas em IA para os próximos 7 dias.

Exemplo de Uso:
const response = await fetch(
  'https://api.terrasphere.com/v1/predictions?lat=-23.5&lng=-46.6',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);
const predictions = await response.json();

Autenticação

API Key

Para requisições simples, use sua chave API no header Authorization:

Authorization: Bearer YOUR_API_KEY

OAuth 2.0

Para aplicações mais complexas, use OAuth 2.0 com fluxo de autorização:

import requests

# Obter token de acesso
auth_response = requests.post('https://api.terrasphere.com/oauth/token', {
    'grant_type': 'client_credentials',
    'client_id': 'YOUR_CLIENT_ID',
    'client_secret': 'YOUR_CLIENT_SECRET'
})

token = auth_response.json()['access_token']

# Usar token nas requisições
headers = {'Authorization': f'Bearer {token}'}
response = requests.get('https://api.terrasphere.com/v1/blooms/current', headers=headers)

SDKs e Bibliotecas

Python SDK

pip install terrasphere-api
from terrasphere import TerraSphereAPI

api = TerraSphereAPI(api_key='YOUR_API_KEY')
blooms = api.get_current_blooms()
print(f"Encontradas {len(blooms)} florações ativas")

JavaScript SDK

npm install @terrasphere/api
import { TerraSphereAPI } from '@terrasphere/api';

const api = new TerraSphereAPI('YOUR_API_KEY');
const blooms = await api.getCurrentBlooms();
console.log(`Encontradas ${blooms.length} florações ativas`);

Java SDK

<dependency>
  <groupId>com.terrasphere</groupId>
  <artifactId>terrasphere-api</artifactId>
  <version>1.0.0</version>
</dependency>
TerraSphereAPI api = new TerraSphereAPI("YOUR_API_KEY");
List<Bloom> blooms = api.getCurrentBlooms();
System.out.println("Encontradas " + blooms.size() + " florações ativas");
