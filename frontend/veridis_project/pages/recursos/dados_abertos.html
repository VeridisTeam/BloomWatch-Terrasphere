Monitoramento de Biotoxinas

Dados de concentração de toxinas marinhas em frutos do mar de programas de monitoramento costeiro.

Cobertura: Costa Atlântica e Pacífica
Frequência: Semanal
Período: 1995 - presente
Formato: CSV, JSON

Documentação da API

Nossa API RESTful fornece acesso programático a todos os conjuntos de dados. Suporta múltiplos formatos de saída e filtros avançados.

GET
/api/v1/chlorophyll
Dados de clorofila satelital
GET
/api/v1/biotoxins
Concentrações de biotoxinas
GET
/api/v1/events
Eventos de florações
GET
/api/v1/environmental
Parâmetros ambientais

Exemplo de Uso

curl -X GET "https://api.terrasphere.com/v1/chlorophyll?lat=-23.5&lon=-46.6&date=2024-01-01" \
  -H "Authorization: Bearer YOUR_API_KEY"
